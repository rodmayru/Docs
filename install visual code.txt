9. Passos per pujar-ho a GitHub
Segueix aquests passos des del Visual Studio Code (assegura't de tenir Git instal·lat):

1) Inicialitzar Repositori:
Obre la terminal a VS Code (Ctrl + ñ).
Escriu: git init

2) Afegir fitxers:
Escriu: git add .

3) Fer el primer commit:
Escriu: git commit -m "Primera versió de l'aplicació d'etiquetes"

4) Crear repositori a GitHub:
Ves a github.com i crea un repositori nou (pots dir-li generador-etiquetes). No l'inicialitzis amb README ni .gitignore des de la web (ja els tens locals).
Copia la URL del repositori (ex: https://github.com/elteuusuari/generador-etiquetes.git).

5) Connectar i Pujar:
A la terminal de VS Code
git remote add origin https://github.com/elteuusuari/generador-etiquetes.git
git branch -M main
git push -u origin main

1. Compila projecte

npm run build

2. Instal·la Capacitor

# Instal·la el nucli de Capacitor
npm i @capacitor/core @capacitor/cli

# Inicialitza Capacitor (posa-li el nom que vulguis a l'app)
npx cap init

3. Afegeix la plataforma Android
npm i @capacitor/android
npx cap add android

4. Sincronitza el codi
npm run build
npx cap copy

5. Genera l'APK amb Android Studio
npx cap open android
