\# Docs



Repositori de documentació comuna per als projectes personals.



Aquest espai serveix per guardar:

\- comandes útils

\- passos d'instal·lació

\- creació d'entorns

\- generació d'APK o AAB

\- notes tècniques

\- configuracions recurrents



\## Estructura



\- `python/` → documentació de Python i entorns virtuals

\- `flutter/` → documentació de Flutter, APK, AAB i builds

\- `android/` → notes d'Android, signing, keystore, configuracions

\- `deploy/` → passos de desplegament

\- `git/` → comandes útils de Git i GitHub



\## Objectiu



Tenir tota la documentació important en un sol lloc, de manera ordenada i reutilitzable entre projectes.



\## Bones pràctiques



\- Escriure cada tema en un fitxer `.md`

\- Posar títols clars

\- Afegir exemples de comandes

\- Indicar rutes de fitxers quan calgui

\- Actualitzar la documentació quan es trobi una solució útil
